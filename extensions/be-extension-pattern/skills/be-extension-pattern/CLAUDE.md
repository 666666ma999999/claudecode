# be-extension-pattern

## トリガーキーワード

以下のキーワードが会話に含まれる場合、このスキルを参照:

- BE新機能追加
- APIエンドポイント追加
- BEエクステンション作成
- バックエンド設計
- HookPoint実装
- BEプラグインアーキテクチャ
- FastAPIエクステンション
- Express/NestJSモジュール設計
- AsyncEventBus
- ExtensionRegistry（BE）

## NOT for

以下の場合はこのスキルを使用しない:

- フロントエンド（React/Next.js）の実装 → `fe-extension-pattern` を使用
- FE+BE連携・APIコントラクト → `fe-be-extension-coordination` を使用
- インフラ・DevOps 設定
- DB スキーマ設計（エクステンション構造に関わらない場合）
- 既存バグの修正（アーキテクチャ変更を伴わない場合）
- パッケージアップデート
