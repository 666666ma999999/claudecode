# fe-be-extension-coordination

## トリガーキーワード

以下のキーワードが会話に含まれる場合、このスキルを参照:

- FE+BE連携
- APIコントラクト
- エクステンション共有設定
- クロスリポ連携
- デプロイ協調
- FE-BEエクステンション整合
- OpenAPI spec管理
- 共有Extension ID
- イベントコントラクト
- compatibility matrix

## NOT for

以下の場合はこのスキルを使用しない:

- FE単体の実装 → `fe-extension-pattern` を使用
- BE単体の実装 → `be-extension-pattern` を使用
- インフラ・DevOps 設定（CI/CD連携以外）
- DB スキーマ設計
- 既存バグの修正（アーキテクチャ変更を伴わない場合）
- パッケージアップデート
