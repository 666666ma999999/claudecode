### カテゴリ別推奨パターン

- **あの人の気持ち**: 連絡待ち解決型(100)、本音深掘り型(79)
- **片思い**: 片思い連絡来る型(43)、名前占い型(20)
- **不倫/W不倫**: 不倫愛本気度型(44)、W不倫決断型(41)、真実シリーズ型(23)
- **復縁**: 復縁可能性診断型(47)
- **結婚**: 結婚相手ヒント型(70)、運命の人発見型(22)
- **SEX**: SEX相性診断型(39)
- **苦しい恋**: 苦しい恋救済型(39)
- **曖昧な関係**: 曖昧な関係決着型(34)
- **汎用**: 連続チェック型(39)、全相性診断型(25)

## 各パターンの詳細構成

詳細は `references/patterns.md` を参照。

## 構成の基本法則

```
序盤（1〜2番目）: 相性・関係性で「土台」
中盤（3〜7番目）: 本音・理由・状況で「深掘り」
終盤（8〜10番目）: 時期・未来・結論で「期待感」
```

## 高売上のポイント

1. **連番形式**: 【本音1】【本音2】... や【真実1】【真実2】...
2. **具体的な時期**: 「○月○日」「30日以内」
3. **二択/判定**: 「〜or〜」「Yes or No」形式
4. **段階的エスカレート**: 軽い話題→深い本音→禁断の願望

## 禁止事項

### 監修者依存キーワード（使用禁止）

| 禁止 | 代替表現 |
|------|----------|
| 霊視、生き霊、念、オーラ | 診断、分析、表と裏 |
| タロット、カード | 真実、チェック |
| ツインレイ、魂 | 特別な絆、運命の相性 |
| 特定監修者名 | 連番（本音1、真実1等） |
| 大予言シリーズ | 予告、展望 |

### 時期依存キーワード（使用禁止）

| 禁止 | 代替表現 |
|------|----------|
| 2026年等の年号 | これから、近いうち |
| 上半期/下半期 | 訪れる変化 |
| 年運 | 人生、自己分析 |

## 分析ワークフロー

### 新商品の見出し設計

1. カテゴリを確認
2. 該当カテゴリの推奨パターンを提示
3. 最高スコアのパターンをベースに提案
4. 商品特徴に合わせてカスタマイズ

### 既存商品の改善

1. 現在の見出し構成を分析
2. 最も近いパターンを特定
3. 高スコアパターンとの差分を提示
4. 改善案を提案

### 売上予測

1. 見出し構成をパターンマッチング
2. 該当パターンのスコアを算出
3. カスタマイズによる補正
4. 最終スコアを提示
