# レイアウトパターン & チャートライブラリ

出典: デジタル庁「ダッシュボードイメージ作成ツールキット」（toolkit.pptx, 14スライド）

## ツールキット構成

| スライド | 内容 |
|---|---|
| 1 | 表紙 |
| 2-3 | ダッシュボードレイアウトテンプレート（空白グリッド） |
| 4 | 完成サンプル（サービスの利用状況ダッシュボード） |
| 5 | チャートライブラリ 1/6 — KPIカード、フィルター、日付範囲 |
| 6-13 | チャートライブラリ 2〜6 — 折れ線、棒、積み上げ、円、表、複合 |
| 14 | 背景/グリッド用素材 |

## ダッシュボードレイアウト基本構造

```
┌──────────────────────────────────────────┐
│ タイトル        名称/ロゴ    00年00月時点 │  ← ヘッダー
├────────┬────────┬────────┬───────────────┤
│ KPI    │ KPI    │ KPI    │ フィルター    │  ← 上段: 全体指標
│ 1,000  │ +100   │ 100%   │              │
├────────┴────────┴────────┤              │
│                          │              │
│   メイングラフ            │  詳細表      │  ← 中段: トレンド
│   （折れ線/棒）           │  市区町村別  │
│                          │  利用率      │
├──────────────────────────┴───────────────┤
│   地域フィルター: 北海道・東北 | 関東 | …  │  ← 下段: 絞込
└──────────────────────────────────────────┘
```

### レイアウト原則（ガイドブック準拠）
- 左上→右下の流れ（全体→部分）
- 16:9画面、縦横6分割グリッド
- 上段: KPIカード（指標値 + 前月比 + 増減表示）
- 中段: メインチャート（時系列推移 or カテゴリ比較）
- 右 or 下段: 詳細テーブル or フィルター
- フィルターは上部か左部に配置

## チャートライブラリ（部品一覧）

### KPIカード
- 指標タイトル
- 主要数値（大きく表示）
- 増減値（+100 / -100、色で正負を区別）
- 比率表示（100%）

### フィルター
- ドロップダウン（「すべて」デフォルト）
- 検索バー
- カテゴリ選択（タブ形式）
- 日付範囲セレクター

### グラフ種類
- 折れ線グラフ（単一系列、複数系列）
- 棒グラフ（縦、横）
- 積み上げ棒グラフ
- 積み上げ面グラフ
- 円グラフ / ドーナッツグラフ
- 複合グラフ（棒 + 折れ線）
- テーブル（行: カテゴリ、列: 数値）

### 背景要素
- グリッド線（配置ガイド用）
- 背景色ブロック（セクション区切り用）

## matplotlib実装時の対応

| ツールキット部品 | matplotlib実装 |
|---|---|
| KPIカード | `ax.text()` + 枠線なしAxes |
| フィルター | 静的画像では不要（注記で代替） |
| 折れ線 | `ax.plot()` |
| 棒グラフ | `ax.bar()` / `ax.barh()` |
| 積み上げ棒 | `ax.bar(bottom=...)` |
| 円グラフ | `ax.pie()`（カテゴリ5以下推奨） |
| テーブル | `ax.table()` or pandas表を画像化 |
| グリッド | `GridSpec(6, 6)` |
