## 🚫 ここに書いてはいけないこと（OPSEC）

| NG | 理由 |
|---|---|
| 「銀行・証券・仮想通貨を管理してる」と明示 | 攻撃者に "狙う価値あり" と告知することになる → 抽象化して「機密ノート」と書く |
| 自分の GitHub アカウント名 / リポ URL | 過去 push の掘り起こしを誘発 |
| 自分の vault の実パス | `<REDACTED_VAULT_PATH>` や `~/MyVault/` に置換 |
| Syncthing デバイス ID / IP / ポート / ホスト名 | ネットワーク攻撃の足がかり |
| disk identifier / Volume UUID | 環境特定情報 |
| パスフレーズの長さ・文字種・覚え方 | 総当たり攻撃のヒント |
| 4層防御の完全構成図 | 防御強度の開示 = 価値ある資産の公告 |
| 構築日時の分単位 | "5月中旬" 程度に丸める |
