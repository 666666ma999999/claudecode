# fe-extension-pattern

## トリガーキーワード

以下のキーワードが会話に含まれる場合、このスキルを参照:

- 新機能追加（FEプロジェクト）
- ページ追加
- ウィジェット追加
- エクステンション作成
- FEアーキテクチャ
- MountPoint 実装
- ext間通信
- Zustand ストア設計（ext内）
- フロントエンド設計
- プラグインアーキテクチャ

## NOT for

以下の場合はこのスキルを使用しない:

- バックエンド（API/サーバー）の実装
- インフラ・DevOps 設定
- DB スキーマ設計
- 既存バグの修正（アーキテクチャ変更を伴わない場合）
- CSS/スタイリングのみの変更
- テストの追加・修正のみ
- パッケージアップデート
